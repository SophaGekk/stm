# Курсовой проект на STM32F103

Система мониторинга температуры на базе STM32F103 с датчиками DS18B20, LM75A и управлением RGB-подсветкой.

Выполнил студент группы 8В32 Гекк Софья

## 📁 Структура проекта

- **GUI/** - Qt приложение для Windows (исходный код)
- **firmware/** - прошивка для STM32 (Keil проект)
- **releases/** - готовые собранные версии программы

## 📥 Установка и запуск

### 1. Установка драйвера ST-LINK
При первом запуске программа предложит установить драйвер. Нажмите "Установить" и следуйте инструкциям.

### 2. Подключение платы STM32
Подключите плату STM32 к компьютеру через USB.

### 3. Запуск приложения
Запустите `stm.exe`

## 🎮 Управление

- **Подключиться** - установка соединения с STM32 по COM-порту
- **Настройка DS18B20** - изменение порогов TH/TL и разрешения
- **Настройка LM75A** - изменение порогов TH/TL
- **RGB Slider** - управление яркостью RGB-подсветки
- **Логирование** - сохранение данных в файл
